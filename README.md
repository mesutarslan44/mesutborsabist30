# mesutborsabist30

mesutborsabist30 is a BIST 30 analysis dashboard built to surface technical signals, market summaries, and action-oriented stock views.
It is designed to make market data easier to scan and interpret in a single interface.

## What It Includes

- BIST 30 focused dashboard views
- Technical analysis oriented stock pages
- Signal and summary driven market monitoring
- Static frontend structure with supporting server-side logic

## Tech Stack

- JavaScript
- HTML
- CSS
- Cloudflare

## Project Goal

This project aims to turn complex market data into a cleaner decision support experience.
The focus is not just data display, but faster reading of signals, momentum, and watchlist opportunities.

## Project Structure

- `site/` contains the frontend files
- `functions/` contains backend or serverless logic used by the project
